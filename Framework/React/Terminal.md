## 1. 프로젝트 생성

```terminal
$ cd 해당폴더
$ create-react-app hello-react
$ cd hello-react
$ npm start(또는 yarn start)
```

## 2. axios  
```terminal
yarn add axios
```

## 3. react-router-dom  
```terminal
yarn add react-router-dom
```

## 4. 리액트 프로젝트 서버에서 실행시키기 위해서 빌드하는 방법  
```terminal
yarn global add server
cd 해당폴더
yarn run build
serve -s build
```
